What is JavaScript

JavaScript is a popular high-level programming language with many uses. It can be used in web development in order to add more dynamic functionality to websites.

HTML vs JavaScript
HTML JavaScript
Definition
HTML is a markup language developed by Tim BernersLee to create web pages. JavaScript is a scripting language that is developed by Brendan Eich to make interactive web pages.
Content Manipulation
Content cannot be changed. It can change content on user actions.
Platform
A web browser is required to display static content. It requires a JavaScript engine to run the code.
JavaScript is embedded in an HTML document. JavaScript makes the web pages dynamic and interactive. CSS is for the styling of HTML elements. HTML is a markup language that gives the primary structure to the web page.
