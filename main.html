Web Development Basics

In order to make a website we combine HTML, CSS, and Javascript. First, Hyper-Text-Markup-Language (aka HTML) is used to define the basic structure of any website. Then we use CSS, a style sheet language, to style the markup language. Finally, we can use JavaScript, a programming language, to add any dynamic properties we would like to have.
